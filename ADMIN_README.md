# 后台管理系统使用说明

## 功能概述

本后台管理系统提供了以下功能：
- 博客文章管理（查看、添加、编辑、删除）
- 仪表盘统计信息
- 简单的登录验证
- 响应式设计，支持移动端访问

## 访问方式

在浏览器中访问：`admin.html`

## 登录信息

默认登录凭据：
- 用户名：`admin`
- 密码：`admin123`

**重要提示：** 为了安全起见，请在首次使用后修改默认密码。

## 主要功能

### 1. 仪表盘
- 查看博客文章总数
- 查看总访问量（示例数据）
- 查看最后更新时间

### 2. 博客管理
- 查看所有博客文章列表
- 添加新文章
- 编辑现有文章
- 删除文章

### 3. 设置
- 修改网站标题
- 修改网站描述

## 注意事项

### 关于数据保存

由于这是一个静态网站，没有后端服务器支持，因此：

1. **博客文章的添加和修改**：
   - 在后台管理系统中添加或修改文章后，系统会提示你手动更新以下文件：
     - `data/blog-posts.json` - 博客文章列表
     - `data/blogs/blogX.md` - 具体的博客内容文件

2. **数据持久化**：
   - 登录状态保存在浏览器的 localStorage 中
   - 文章修改仅在前端显示，需要手动更新到实际文件中

### 安全建议

1. 修改默认登录凭据
2. 不要将 `admin.html` 文件部署到生产环境（除非你添加了适当的安全措施）
3. 考虑使用 GitHub Actions 或其他自动化工具来处理文件更新

## 自定义修改

### 修改登录凭据

编辑 `js/admin.js` 文件，找到以下代码：

```javascript
if (username === 'admin' && password === 'admin123') {
```

修改为你的用户名和密码。

### 添加新功能

1. 在 `admin.html` 中添加相应的 HTML 结构
2. 在 `css/admin.css` 中添加样式
3. 在 `js/admin.js` 中添加功能逻辑
